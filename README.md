# BookTracker

A personal book reading tracker built with Next.js, Zustand, and Tailwind CSS.

## Features

- **Bookshelf** - Browse and filter your book collection by status (reading, unread, completed)
- **Reading Rounds** - Track multiple reading rounds per book, switch between current and historical rounds
- **Chapter Scheduling** - Schedule chapters for today, tomorrow, or next week
- **Progress Tracking** - Visual progress bars and per-chapter completion status
- **Today's Reading List** - Dashboard view grouped by book with full chapter paths
- **Author Management** - Organize books by author with dedicated author pages
- **Local Storage** - All data persists in the browser via localStorage

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- [React 19](https://react.dev/)
- [Zustand](https://zustand-demo.pmnd.rs/) (state management)
- [Tailwind CSS 4](https://tailwindcss.com/)
- TypeScript

## Getting Started

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm start
```
